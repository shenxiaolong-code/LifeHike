
# 目标
持续为高血脂所困扰，2025.11.22日实施断食调节

# 记录
- 断食期间只喝水，每天补充电解质(一粒康比特丸)，维生素（善存银片丸）。
- 前二天比较饥饿，第三天开始饥饿感消失，但有头晕现象，补充电解质后缓解。
- 第四天有口腔溃疡现象，服用维生素（善存银片）后消失。
- 第五天，体感良好，但是头晕变频繁，补充电解质后可短暂缓解，但很快会重新头晕。
- 断食减脂120小时后，体能依旧感觉良好，肌肉有力，体感轻快，可以运动，有锻炼的欲望。
- 在16度气温情况下，能够穿单衣自由活动而仍然处于舒适区。

# 断食恢复
- 第一天只吃流食：稀饭或者燕麦粥。
- 第二天可吃干饭，菜不可油腻。
- 第三天可正常饮食

# 结论
- 断食可有效改善身体体感健康状况。
- 断食需要准备补盐液，否则单纯电解质补足不够。

# 后遗症
- 断食120小时结束后，如果无耐力活动，体感无影响。
- 断食20天内，极其影响长途爬山等耐力活动，爬山过程中有严重的动力不足感觉，感觉需要进补油腻食物。
- 断食后相当长的时间内无法进行耐力活动。

# 经验(长断食准备)
- 去药店买补盐液，准备好电解质和维生素，食盐，油菜、菠菜，青菜、白菜。（结束断食后十天内不要参加高强度爬山活动）
- 断食初期（1-2天）：只补充水，每天一粒（维生素，电解质）
- 中期（2-3天）：加补充食盐水
- 后期（4-5...天）：加补盐液，纯菜汁（煮熟过滤，现做现喝）
