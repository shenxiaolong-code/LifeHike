
2025.11.11日住宿重庆奉节县城，第二天打车去三峡之巅入口瞿塘峡白帝城入口。

# 爬山行程
- 全程10公里，耗时6小时。
- 2025.11.12日,短裤短袖，背着30斤65升登山包，手提1斤半葡萄，两手桔子出发，购买门票后，进入白帝城风景区忠义广场爬山起点。
简单游览白帝庙，杜甫亭后，乘船过江，到达夔门，此后为正式的离开平路，登山开始。
登山开始，遇到两个巡山工作人员，看到我外挂的水果，强烈要求我把水果放入包内，或者用衣服掩盖起来。
我听从了建议把水果放入包内，后来证明了这个顺从非常明智，因为后面第一次见到敢抢游客食物的野猴，一个女游客惨遭掠夺，外套上留下道道抓痕。
前面一路没有岔路，较为轻松，尤其是六十六道拐之前，基本为山间休闲道路，起伏不大，爬升较小,直到进入赤甲楼徒步区域。
从六十六道拐开始，才是正式的爬升，六十六道拐段道路不错，中等强度的爬山。
过了六十六道拐外，才爬山强度逐渐提升，直到最后一公里的一座山，是全程的最大爬升路段。
因为前面仅靠水果支撑，后期能量补给耗尽，爬得异常艰难。


# 消费
- 三峡之巅消费比正常的物价贵，并且补给较缺，前半程无补给，后半程有卖水卖泡面的。
- 三峡之巅需要买两次门票，白帝城门票和赤甲楼门票(赤甲楼必须要在下午13：00之前赶到才能进入)

# 体验
- 小心野猴抢夺食物水果。需要备肉类路餐，光水果后期体力跟不上。
- 最后一公里，才是爬升的艰难之处：体力末端，最大的爬升，全程80%的爬升都在最后一公里。
- 山上山下温差较大，山下16度，山上8度，有雾无风
