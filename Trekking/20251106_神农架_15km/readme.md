
2025.11.06 日连续四天，久闻神农架，今日终得一见

# 爬山行程
- 2025.11.06日，官门山，14.36公里。单人独自出行，翻山进入封锁区域，第一次见到绝品植物，娃娃鱼，山区塌方现场，亲身经历了负氧粒子高达4016的神农泉。
咳嗽了20年，到达神农泉开始，完全不再咳嗽了，并且余下的几天也完全不咳嗽了。
- 2025.11.07日，途经神农谷，神农顶，浏览九大湖，神农架的精髓之所在，可惜时间太短，在前面浪费了太多的时间，在最后一湖只好潦草路过。
- 2025.11.08日，步行至神农坛，跪拜先祖。
- 2025.11.09日，乘车去天生桥及天燕景区，一睹珍稀的金丝血洞穴，虎谭，鹰谭，牛鼻子洞，野人洞，云台，彩虹桥。

# 体验
- 人间仙境，值得久居。
- 如果神农架有年票，就相当的值得，可以经常去几天景区
- 神农架主要贵在交通。物价也偏贵。


# 个人建议：
- 易不赶时间，慢慢欣赏
